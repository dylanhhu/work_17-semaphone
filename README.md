# work_17-semaphone
Dylan Hu - Work 17: Semaphone!

## Compilation
Compile using `make`.

## Usage
### `control`
The controller for the game. Sets up and cleans up the game.

#### Arguments
* `-c --create`
    * Sets up the game
* `-r --remove`
    * Cleans up the game. Leaves the game log file intact.

### `game`
The actual game. Prompts are given by the program.
